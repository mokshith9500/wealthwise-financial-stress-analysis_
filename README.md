WealthWise: Financial Stress & Banking System Analysis
Overview

This project analyzes banking system indicators and financial stress using unsupervised machine learning methods.

Methods applied:

Principal Component Analysis (PCA)

K-Means Clustering

Hierarchical Clustering

DBSCAN

Association Rule Mining (Apriori Algorithm)

Data Source

All economic indicators were collected from the Federal Reserve Economic Data (FRED) database using the official FRED API.

Data Processing

Time-series merged on date

Missing values handled using forward fill

Yield spread feature engineered

StandardScaler used prior to PCA and clustering

Discretization performed for ARM

Repository Structure

Notebook contains full analysis and modeling

Raw dataset before cleaning included

Cleaned dataset used for modeling included

Reproducibility

All code and datasets are provided to ensure transparency and repeatability.
